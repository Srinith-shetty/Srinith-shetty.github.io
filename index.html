Smart Home IoT System

By Srinith Shetty

📌 Project Overview

This project demonstrates an IoT-based home automation system using the ESP8266 NodeMCU and a 2-channel relay module. It allows users to control devices remotely using the Blynk app over Wi-Fi.

🔧 Components Used

  • ESP8266 NodeMCU
  • 2-Channel Relay Module
  • 2 LEDs
  • 9V Battery
  • Breadboard
  • Jumper Wires
  • USB Cable
  • Wi-Fi Hotspot

🔌 Circuit Connections

Component Connection
Relay IN1 D1 (GPIO5)
Relay IN2 D2 (GPIO4)
Relay VCC 3.3V
Relay GND GND

💻 Software & Libraries

  • Arduino IDE
  • ESP8266 Board Package
  • Blynk Library

📱 Blynk Setup

  • Create a device in Blynk Cloud
  • Add two buttons:
    • V1 → Relay 1
    • V2 → Relay 2
  • Set mode to Switch

▶️ How to Run

git clone https://github.com/<your-username>/Smart-home-Project.git
  • Open in Arduino IDE
  • Update Wi-Fi credentials
  • Upload code to NodeMCU
  • Control using Blynk app

⚙️ Working Principle

The NodeMCU connects to Wi-Fi and communicates with the Blynk app. Based on user input, it controls GPIO pins which trigger relays to switch devices ON/OFF.
